01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다.

변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 변경

변수는 데이터 저장과 변경이 가능합니다. 변수의 선언 후 변수변경 방법은 아래와 같습니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jQuery";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과 확인하기
300
400
jQuery

03. 변수 : 데이터 저장 + 변경 + 추가

변수는 데이터 저장과 변경이 또 동적으로 추가가 가능합니다. 이미 선언된 변수에 '+'연산자를 사용하여 추가하는 방법은 아래와 같습니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jQuery";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과 확인하기
400
600
javascriptjQuery

05-1. 배열 : 데이터 저장(여러개) : 표현방법1

배열의 표현 방법중 한가지로 여기서 new Array()는 배열 객체의 생성자를 호출합니다. 생성자를 호출하면서 인자를 전달하지 않으면, 빈 배열이 생성됩니다. 이러한 방식은 배열을 생성할 때 초기 크기나 초기 값을 지정할 필요가 없을 때 유용합니다. 예시는 아래와 같습니다.

{
    const arr = new Array();
    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0])
    console.log(arr[1])
    console.log(arr[2])
}
결과 확인하기
100
200
javascript

05-2. 배열 : 데이터 저장(여러개) : 표현방법2

배열의 표현 방법중 한가지로 여기서 new Array()는 배열 객체의 생성자를 호출합니다. 생성자를 호출하면서 인자를 전달함으로서, 값이 존재하는 배열이 생성됩니다. 이러한 방식은 배열을 생성할 때 초기 크기나 초기 값을 지정할 필요가 있을 때 유용합니다. 예시는 아래와 같습니다.

{
    const arr = new Array(100, 200, "javascript");

    console.log(arr[0])
    console.log(arr[1])
    console.log(arr[2])
}
결과 확인하기
100
200

05-3. 배열 : 데이터 저장(여러개) : 표현방법3

JavaScript에서 배열을 생성하는 더 간단하고 일반적인 방법입니다. []는 배열 리터럴 표기법입니다. 배열 리터럴은 대괄호([]) 안에 요소를 나열하여 배열을 생성합니다. 만약 요소가 없는 경우에는 빈 배열이 생성됩니다. 이 방법은 배열을 생성하고 초기화할 때 매우 자주 사용됩니다. 예시는 아래와 같습니다.

{
    const arr = [];

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0])
    console.log(arr[1])
    console.log(arr[2])
}
결과 확인하기
100
200
javascript

JavaScript에서 배열을 생성하는 더 간단하고 일반적인 방법입니다. []는 배열 리터럴 표기법입니다. 배열 리터럴은 대괄호([]) 안에 요소를 나열하여 배열을 생성합니다. 만약 요소가 있는 경우에는 값이 존재하는 배열이 생성됩니다. 이 방법은 배열을 생성하고 초기화할 때 매우 자주 사용됩니다. 예시는 아래와 같습니다.

{
    const arr = [100, 200, "javascript"];

05-5. 배열 : 데이터 저장(여러개) : 표현방법5

05-6. 배열 : 데이터 저장(여러개) : 표현방법6

다차원 배열은 배열 내에 또 다른 배열이 포함된 배열을 의미합니다. 즉, 배열 안에 배열이 중첩되어 있는 형태입니다. 2차원 배열은 행과 열로 이루어진 행렬(matrix)과 같은 형태를 가집니다. 각 행은 배열의 요소이고, 각 요소는 또 다른 배열을 가지고 있습니다.이러한 형태의 다차원 배열은 데이터를 구조화하고 계층적으로 표현할 때 유용합니다. 예시는 아래와 같습니다.

{
    const arr = [100, 
        [200, 300, 
            [400, 500, 
                [600]
            ]
        ], 
        ["javascript", "jQuery"]
    ];

    console.log(arr[0])
    console.log(arr[1][0])
    console.log(arr[1][1])
    console.log(arr[2][0])
    console.log(arr[2][1])
    console.log(arr[1][2][2][0])
}
결과 확인하기
100
200
300
javascript
jQuery
600

05-7. 배열 : 데이터 저장(여러개) : 표현방법7

이 코드는 배열 해체(destructuring)를 사용하여 배열 arr의 각 요소를 변수 a, b, c에 할당하는 예시를 보여줍니다.이러한 배열 해체는 배열에서 원하는 요소를 더 쉽게 추출하여 변수에 할당할 때 유용합니다. 예시는 아래와 같습니다.

{
    const arr = [100, 300, "javascript"];
    const [a, b, c] = arr;

    console.log(a)
    console.log(b)
    console.log(c)
}
{
    const arr = ['user', 'rank', [300, 400]];
    const [a, b, [x, y]] = arr;

    console.log(a)
    console.log(b)
    console.log(x)
    console.log(y)
}

05-7. 배열 : 데이터 저장(여러개) : 표현방법8

이 코드는 배열을 조합하고 확장하는 예시를 보여줍니다. 결과적으로 arr3 배열은 숫자 100, 300, "javascript", 500을 포함하게 됩니다. 이러한 배열 조합과 확장은 여러 배열을 하나로 결합하거나 새로운 요소를 추가할 때 유용합니다.

{
    const arr1 = [100, 300, "javascript"];
    const arr2 = [200, 400, "jQuery"];
    const arr3 = [...arr1, 500];

    console.log(...arr1)
    console.log(...arr2)
    console.log(...arr3)
    console.log(...arr1, ...arr2)
    console.log(...arr1, ...arr2, ...arr3)
}
결과 확인하기
'javascript'
'jQuery'
'javascript' 500
'javascript' 200 400 'jQuery'
'javascript' 200 400 'jQuery' 100 300 'javascript' 500
