01. 변수 : 데이터 저장

자바스크립트에서 변수는 값을 저장하고 참조하는데 사용합니다.

변수를 사용하면 데이터를 저장하고 나중에 해당 데이터를 재사용 할 수 있습니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 변경

변수는 데이터 저장과 변경이 가능합니다. 변수의 선언 후 변수변경 방법은 아래와 같습니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x = 300;
    y = 400;
    z = "jQuery";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과 확인하기
300
400
jQuery

03. 변수 : 데이터 저장 + 변경 + 추가

변수는 데이터 저장과 변경이 또 동적으로 추가가 가능합니다. 이미 선언된 변수에 '+'연산자를 사용하여 추가하는 방법은 아래와 같습니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jQuery";

    console.log(x)
    console.log(y)
    console.log(z)
}
결과 확인하기
400
600
javascriptjQuery

05-1. 배열 : 데이터 저장(여러개) : 표현방법1

배열의 표현 방법중 한가지로 여기서 new Array()는 배열 객체의 생성자를 호출합니다. 생성자를 호출하면서 인자를 전달하지 않으면, 빈 배열이 생성됩니다. 이러한 방식은 배열을 생성할 때 초기 크기나 초기 값을 지정할 필요가 없을 때 유용합니다. 예시는 아래와 같습니다.

{
    const arr = new Array();
    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0])
    console.log(arr[1])
    console.log(arr[2])
}
결과 확인하기
100
200

    console.log(arr[2])

05-6. 배열 : 데이터 저장(여러개) : 표현방법6

다차원 배열은 배열 내에 또 다른 배열이 포함된 배열을 의미합니다. 즉, 배열 안에 배열이 중첩되어 있는 형태입니다. 2차원 배열은 행과 열로 이루어진 행렬(matrix)과 같은 형태를 가집니다. 각 행은 배열의 요소이고, 각 요소는 또 다른 배열을 가지고 있습니다.이러한 형태의 다차원 배열은 데이터를 구조화하고 계층적으로 표현할 때 유용합니다. 예시는 아래와 같습니다.

{
    const arr = [100, 
        [200, 300, 
            [400, 500, 
                [600]
            ]
        ], 
        ["javascript", "jQuery"]
    ];

    console.log(arr[0])
    console.log(arr[1][0])
    console.log(arr[1][1])
    console.log(arr[2][0])
    console.log(arr[2][1])
    console.log(arr[1][2][2][0])
}

05-7. 배열 : 데이터 저장(여러개) : 표현방법8

이 코드는 배열을 조합하고 확장하는 예시를 보여줍니다. 결과적으로 arr3 배열은 숫자 100, 300, "javascript", 500을 포함하게 됩니다. 이러한 배열 조합과 확장은 여러 배열을 하나로 결합하거나 새로운 요소를 추가할 때 유용합니다.

    console.log(...arr1, ...arr2)
    console.log(...arr1, ...arr2, ...arr3)
}

07-1. 파일 : 데이터 교환(json)

이 코드는 JavaScript에서 JSON 데이터를 생성하고 파싱하는 과정을 보여줍니다. 먼저, 문자열 형태의 JSON 데이터를 객체로 파싱하고, 이후 JavaScript 객체를 JSON 문자열로 변환하는 과정을 보여줍니다. JSON.parse() 함수를 사용하여 문자열 형태의 JSON 데이터를 JavaScript 객체로 변환하고, JSON.stringify() 함수를 사용하여 JavaScript 객체를 JSON 문자열로 변환합니다. 결과적으로는 JSON 데이터를 객체로 파싱하고 객체를 다시 JSON 문자열로 변환한 것을 확인할 수 있습니다

{
    const jsonString = '{"name":"Cho", "age":30}';
    const jsonObject = JSON.parse(jsonString);
    console.log(jsonObject.name)
    console.log(jsonObject.age)
}
{
    
    const person = { name: "Cho", age: 30 };
    const jsonString = JSON.stringify(person);

    console.log(jsonString)
}

30

++ 연산자

JavaScript에서 연산자는 값을 계산하고 조작하는 데 사용됩니다. 다양한 종류의 연산자가 있으며, 이를 통해 산술, 할당, 비교, 논리 등의 작업을 수행할 수 있습니다.

연산자 설명
산술 연산자 숫자를 계산하는 데 사용됨 (+, -, *, /, %, ++, --)
할당 연산자 변수에 값을 할당하는 데 사용됨 (=, +=, -=, *=, /=, %=, <<=,>>=, >>>=, &=, ^=, |=)
비교 연산자 두 값의 관계를 비교하여 true 또는 false를 반환함 (==, ===, !=, !==, <,>, <=,>=)
논리 연산자 논리적인 조건을 평가하여 true 또는 false를 반환함 (&&, ||, !)
비트 연산자 이진수 값에 대한 비트 단위 연산을 수행함 (&, |, ^, ~, <<,>>, >>>)
조건(삼항) 연산자 조건에 따라 값을 반환함 (condition ? expr1 : expr2)
타입 연산자 피연산자의 타입을 반환함 (typeof)
그 외 연산자 콤마 연산자, 점 연산자 등 (,, .)

++ 자료형

JavaScript 자료형 연산자 중 typeof는 피연산자의 데이터 유형을 문자열 형태로 반환하며, instanceof는 객체가 특정 클래스의 인스턴스인지를 확인하여 불리언 값을 반환합니다.

연산자 설명 예시
typeof 피연산자의 데이터 유형을 나타내는 문자열을 반환함 typeof 42 → "number"
typeof "hello" → "string"
typeof true → "boolean"
typeof undefined → "undefined"
typeof null → "object"
typeof [] → "object"
typeof {} → "object"
typeof function() {} → "function"
instanceof 객체가 특정 클래스의 인스턴스인지 확인함 var obj = {};
obj instanceof Object → true
