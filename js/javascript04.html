01. if문(조건식)

if문은 주어진 조건식이 참이면 특정 코드를 실행하고, 거짓이면 다른 코드를 실행합니다. 

{
    //true "문자열" 숫자 [] {}
    //false ""(빈문자열) 0 null undefined

    if ("조건") {
        console.log("실행되었습니다.(true)");
    } else {
        console.log("실행되었습니다.(false)");
    }
}
결과 확인하기
실행되었습니다.(true)

02. 다중 if문

다중 if문은 여러 개의 조건을 순차적으로 검사하여 조건이 참인 경우 해당 코드 블록을 실행하는 제어 구조입니다. 

{
    const num = 100;

    if (num == 90) {
        console.log("실행되었습니다.(90)");
    } else if (num == 100) {
        console.log("실행되었습니다.(100)");
    } else if (num == 110) {
        console.log("실행되었습니다.(110)");
    } else {
        console.log("실행되었습니다.");
    }
}
결과 확인하기
실행되었습니다.(100)

03. 중첩 if문

중첩 if문은 if문 내부에 또 다른 if문을 사용하여 더 복잡한 조건을 처리하는 제어 구조입니다. 

{
    const num = 100;

    if (num == 100) {
        console.log("실행되었습니다.1");
        if (num == 100) {
            console.log("실행되었습니다.2");
            if (num == 100) {
                console.log("실행되었습니다.3");
            }
        }
    } else {
        console.log("실행되었습니다.4");
    }
}
결과 확인하기
실행되었습니다.1
실행되었습니다.2
실행되었습니다.3

04. if문 생략 & 삼항 연산자

삼항 연산자는 조건식을 평가하여 참 또는 거짓에 따라 다른 값을 반환하는 연산자입니다. 일반적으로 조건식 ? 참일 때 값 : 거짓일 때 값 형식으로 사용됩니다. 

{
    const num = 100;

    if (num == 100) {
        console.log("true");
    }
    // 생략
    if (num == 100) console.log("true");

    if (num == 100) {
        console.log("true");
    } else {
        console.log("false");
    }

    //삼항 연산자(조건식 연산자)
    num == 100 ? console.log("true") : console.log("false");
}

05. switch문

switch문은 프로그래밍에서 하나의 변수의 값에 따라 여러 가지 경우를 처리하는 제어 구조입니다. 변수의 값에 따라 해당하는 case 문을 실행하고, 일치하는 경우에는 해당 case 문 이후의 코드를 실행합니다. 

{
    const num = 100;

    switch (num) {
        case 90:
            console.log("실행되었습니다.(90)");
            break;
        case 95:
            console.log("실행되었습니다.(95)");
            break;
        case 100:
            console.log("실행되었습니다.(100)");
            break;
        default:
            console.log("실행되었습니다.");
    }
}
결과 확인하기
실행되었습니다.(100)

06. while문

while문은 주어진 조건식이 참일 때에 한하여 코드 블록을 반복적으로 실행하는 반복문입니다. 조건식이 거짓이 될 때까지 반복됩니다. 

{
    let num = 1;

    while (num <= 5) {
        console.log(num);
        num++;
    }
}

07. do while문

do while문은 코드 블록을 먼저 실행한 후에 조건을 검사하여 조건이 참인 경우에는 코드 블록을 반복 실행하는 반복문입니다. 최소한 한 번은 코드 블록이 실행됩니다. 

{
    let num = 1;

    do {
        console.log(num);
        num++;
    } while (num <= 5);
}

08. for문

for문은 초기화, 조건식, 증감식을 이용하여 지정된 횟수만큼 반복하여 코드 블록을 실행하는 반복문입니다. 반복 횟수가 정해져 있을 때 주로 사용됩니다. 

{
    for (let i = 1; i <= 5; i++) {
        console.log(i);
    }
}

09. 중첩 for문

중첩 for문은 하나의 for문 내부에 또 다른 for문을 포함하여 여러 번 반복하는 반복문 구조입니다. 이를 통해 2차원 이상의 데이터 구조를 처리하거나 이차원 배열을 순회할 수 있습니다. 

{
    for (let i = 1; i <= 3; i++) {
        for (let j = 1; j <= 3; j++) {
            console.log(i, j);
        }
    }
}

10. break문

break문은 반복문(for문, while문 등) 내에서 실행되며, 해당 반복문을 즉시 종료하고 빠져나가는 역할을 합니다. 주로 특정 조건이 충족되었을 때 반복을 중단할 때 사용됩니다. 

{
    for (let i = 1; i <= 10; i++) {
        if (i == 5) {
            break;
        }
        console.log(i);
    }
}

11. continue문

continue문은 반복문(for문, while문 등) 내에서 실행되며, 현재 반복을 중단하고 다음 반복으로 넘어가는 역할을 합니다. 따라서 continue문 이후의 코드는 실행되지 않고 다음 반복이 시작됩니다. 

{
    for (let i = 1; i <= 10; i++) {
        if (i == 5) {
            continue;
        }
        console.log(i);
    }
}
