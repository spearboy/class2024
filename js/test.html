01. 1부터 100까지 출력하기(for, while, do while)

반복문에서 for, while, do while 3가지 문법 입니다.

{
    for (let i = 0; i < 100; i++) {
        console.log(i + 1)
    }
}
{
    let num = 0;
    while (num < 100) {
        console.log(num + 1)
        num++;
    }
}
{
    let num = 0;
    do {
        console.log(num + 1)
        num++;
    } while (num < 100)
}
결과 확인하기
for : 1,2,3,4,5....100
while : 1,2,3,4,5....100
do while : 1,2,3,4,5....100

02. 1부터 100까지 짝수만 출력하기(for)

반복문의 홀수또는 짝수만 출력하는 예제입니다..

{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 0) {
            console.log(i)
        }
    }
}
{
    for (let i = 1; i <= 100; i++) {
        if (i % 2 == 1) {
            console.log(i)
        }
    }
}
결과 확인하기
1번 : 2,4,6....100
1번 : 1,3,5....99
