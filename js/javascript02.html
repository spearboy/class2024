20. 객체 : 데이터 불러오기 : 객체 구조 분해 할당

객체 구조 분해 할당은 JavaScript에서 객체의 속성을 추출하여 개별 변수로 할당하는 기능입니다. 이를 통해 객체에서 필요한 속성만 추출하여 간단한 변수로 만들 수 있습니다. 이는 객체의 특정 속성에 직접 접근하는 것보다 코드를 더 간결하고 가독성 있게 만들어 줍니다.

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    };
    const { a, b, c } = obj;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    // 기본 값 할당 
    const obj = {
        a: 100,
        b: 200
    };
    const { a, b, c = "javascript" } = obj;

    console.log(a);
    console.log(b);
    console.log(c);
}
{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    };
    const { a: x, b: y, c: z } = obj;

    console.log(x);
    console.log(y);
    console.log(z);
}
{
    const obj = {
        a: 100,
        b: { c: 200, d: "javascript" }
    };
    const { a, b: { c, d } } = obj;

    console.log(a)
    console.log(c)
    console.log(d)
}

21. 객체 : 데이터 불러오기 : 객체 펼침연산자

객체 펼침 연산자는 JavaScript에서 객체를 확장하거나 병합할 때 사용됩니다. 이 연산자는 기존 객체의 속성을 새로운 객체에 펼쳐 넣어주는 역할을 합니다. 위 코드에서는 obj1과 obj2 객체의 속성을 펼쳐서 새로운 객체로 만들고 있습니다. merge1은 두 개의 객체를 단순히 병합한 것이며, merge2는 추가적으로 새로운 속성을 포함하여 객체를 확장한 것을 보여줍니다.

{
    const obj1 = { a: 100, b: 200 };
    const obj2 = { c: 300, d: 400 };
    const merge1 = { ...obj1, ...obj2 };
    const merge2 = { ...obj1, x: "javascript", ...obj2, y: 'jquery' };
    console.log(obj1)
    console.log(merge1)
    console.log(merge2)
}
결과 확인하기
{a: 100, b: 200}
{a: 100, b: 200, c: 300, d: 400}
{a: 100, b: 200, x: 'javascript', c: 300, d: 400, …}

22. 서버 데이터 불러오기 : XMLHttpRequest

이 코드는 XMLHttpRequest 객체를 사용하여 주어진 URL에서 JSON 형식의 데이터를 비동기적으로 가져옵니다. 데이터를 성공적으로 가져오면 JSON 형식으로 변환하고 콘솔에 출력하며, 가져오는 동안 오류가 발생하면 오류 메시지를 콘솔에 출력합니다. xhr.send();는 요청을 서버로 보내는 역할을 합니다.

{
    var xhr = new XMLHttpRequest();
    xhr.open("GET", "https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json", true);

    xhr.onreadystatechange = function () {
        if (xhr.readyState === 4 && xhr.status === 200) {
            var responseData = xhr.responseText;
            var jsonData = JSON.parse(responseData);

            console.log(jsonData);
        } else if (xhr.readyState === 4) {
            console.error("데이터 불러오기 오류: " + xhr.status);
        }
    };

    xhr.send();
}

23. 서버 데이터 불러오기 : fetch API

이 코드는 주어진 URL에서 JSON 형식의 데이터를 가져와서 콘솔에 출력합니다. 데이터를 성공적으로 가져오면 콘솔에 데이터를 출력하고, 가져오는 동안 오류가 발생하면 해당 오류 메시지를 출력합니다.

{
    fetch("https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json").then(response => {
        return response.json();
    }).then(data => {
        console.log(data)
    }).catch(error => {
        console.log('데이터 불러오기 오류', error)
    })
}
