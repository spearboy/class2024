01. 함수 : 선언적 함수

선언적 함수(Declarative Function)는 원하는 결과물을 정의하고 해당 결과물을 생성하기 위한 과정을 명시하지만, 어떻게 그 결과를 얻을지에 대한 구체적인 작업 흐름을 직접 명시하지 않는 함수입니다. 

{
    function func() {
        console.log("01. 함수가 실행되었습니다.");
    }
    func();
}

02. 함수 : 익명 함수

익명 함수(Anonymous Function)는 이름이 없는 함수를 말합니다. 일반적으로 함수를 정의할 때 함수의 이름을 명시하지 않고, 함수를 변수에 할당하거나 다른 함수의 매개변수로 전달할 때 사용됩니다. 

{
    const func = function () {

    }
    func();
}

03. 함수 : 매개 함수

"매개 함수(Parameterized Function)"는 함수가 실행될 때 외부로부터 값을 받아들이는 데 사용되는 매개변수를 가지고 있는 함수를 의미합니다. 매개 함수는 함수를 호출할 때 인자(argument)를 전달하여 함수 내부로 값을 전달할 수 있게 합니다. 

{
    // 선언적 함수
    function func(str) {
        console.log(str);
    }
    func("03. 함수가 실행되었습니다.");

    //익명 함수
    const func1 = function (str) {
        console.log(str);
    }
    func1("03. 함수가 실행되었습니다.")
}

04. 함수 : 리턴값 함수

리턴값 함수(Return Value Function)는 함수가 실행된 후에 결과를 반환하는 데 사용되는 함수입니다. 이 함수는 일련의 작업을 수행하고 그 결과를 호출 지점으로 반환합니다. 

{
    //선언적 함수
    function func() {
        return "04. 함수가 실행되었습니다.";
    }
    console.log(func());

    // 익명 함수
    const func1 = function () {
        return "04. 함수가 실행되었습니다.";
    }
    console.log(func1());
}

10. 화살표 함수 : 매개변수 + 리턴값 함수

화살표 함수는 간결한 문법으로 매개변수를 받아들이고 결과를 반환하는 익명 함수를 정의하는 데 사용됩니다. 이 함수는 => 연산자를 사용하여 매개변수와 결과를 나타내며, 단일 표현식을 포함하는 경우 중괄호 및 return 문을 생략할 수 있습니다. 

{
    //선언적 함수
    func = (str) => {
        return str;
    }
    console.log(func("10. 함수가 실행되었습니다."));

    //익명 함수
    const func1 = (str) => {
        return str;
    }
    console.log(func1("10. 함수가 실행되었습니다."));

    //매개변수 하나이면 괄호 생략 가능
    const func2 = str => {
        return str;
    }
    console.log(func2("10. 함수가 실행되었습니다."));

    //리턴 생략
    const func3 = str => str;
    console.log(func3("10. 함수가 실행되었습니다."));

    //선언적 함수(가독성X)
    func4 = str => str;
    console.log(func4("10. 함수가 실행되었습니다."));
}

17. 함수 유형: 객체 생성자 함수

주어진 JavaScript 코드에서는 객체 생성자 함수를 활용하여 객체를 생성하고, 해당 객체의 프로토타입에 함수를 추가하여 메서드를 정의하는 예제입니다. 생성된 객체는 이후에 해당 메서드를 호출하여 정보를 출력합니다. 

{

    const info2 = new Func(20, "함수가 실행되었습니다.");

    // 호출
    info1.result();
    info2.result();
}
결과 확인하기

18. 함수 유형: 프로토타입 함수

프로토타입 함수는 객체의 프로토타입에 메서드를 추가하여 모든 해당 객체의 인스턴스에서 공유하여 사용할 수 있게 만드는 것입니다. 

{
    // 함수 정의
    function Func(num, str) {
        this.num = num;
        this.str = str;
    }

    // 메서드 추가
    Func.prototype.result = function () {
        console.log(`${this.num}. ${this.str}`);
    }

    // 인스턴스 생성
    const info1 = new Func(21, "함수가 실행되었습니다.");
    const info2 = new Func(22, "함수가 실행되었습니다.");

    //호출
    info1.result();
    info2.result();
}
결과 확인하기
21. 함수가 실행되었습니다.
22. 함수가 실행되었습니다.

19. 함수 유형: 객체 리터럴 함수

주어진 JavaScript 코드는 객체 리터럴을 사용하여 함수를 정의하고, 해당 객체에 메서드를 추가하는 예제입니다. 객체를 생성할 때 함수를 정의하고, 이후에는 해당 메서드를 호출하여 정보를 출력합니다. 

{
    // 함수 정의
    function Func(num, str) {
        this.num = num;
        this.str = str;
    }

    // 메서드 추가
    Func.prototype = {
        result1: function () {
            console.log(`${this.num}. ${this.str}`);
        },
        result2: function () {
            console.log(`${this.num}. ${this.str}`);
        }
    }

    // 인스턴스 생성
    const info1 = new Func(23, "함수가 실행되었습니다.");
    const info2 = new Func(24, "함수가 실행되었습니다.");

    //호출
    info1.result1();
    info2.result2();
}
결과 확인하기
23. 함수가 실행되었습니다.
24. 함수가 실행되었습니다.
