01. 함수 : 선언적 함수

선언적 함수(Declarative Function)는 원하는 결과물을 정의하고 해당 결과물을 생성하기 위한 과정을 명시하지만, 어떻게 그 결과를 얻을지에 대한 구체적인 작업 흐름을 직접 명시하지 않는 함수입니다. 

{
    function func() {
        console.log("01. 함수가 실행되었습니다.");
    }
    func();
}

02. 함수 : 익명 함수

익명 함수(Anonymous Function)는 이름이 없는 함수를 말합니다. 일반적으로 함수를 정의할 때 함수의 이름을 명시하지 않고, 함수를 변수에 할당하거나 다른 함수의 매개변수로 전달할 때 사용됩니다. 

{

03. 함수 : 매개 함수

"매개 함수(Parameterized Function)"는 함수가 실행될 때 외부로부터 값을 받아들이는 데 사용되는 매개변수를 가지고 있는 함수를 의미합니다. 매개 함수는 함수를 호출할 때 인자(argument)를 전달하여 함수 내부로 값을 전달할 수 있게 합니다. 

{
    // 선언적 함수
    function func(str) {
        console.log(str);
    }
    func("03. 함수가 실행되었습니다.");

    //익명 함수
    const func1 = function (str) {
        console.log(str);
    }
    func1("03. 함수가 실행되었습니다.")
}

04. 함수 : 리턴값 함수

리턴값 함수(Return Value Function)는 함수가 실행된 후에 결과를 반환하는 데 사용되는 함수입니다. 이 함수는 일련의 작업을 수행하고 그 결과를 호출 지점으로 반환합니다. 

{
    //선언적 함수
    function func() {
        return "04. 함수가 실행되었습니다.";
    }
    console.log(func());

    // 익명 함수
    const func1 = function () {
        return "04. 함수가 실행되었습니다.";
    }
    console.log(func1());
}

{

        return str;

    //호출

}

20. 함수: 즉시실행 함수

즉시 실행 함수(IIFE)는 함수를 정의하고 즉시 실행하는 JavaScript의 패턴입니다. 이를 통해 스코프를 보호하고 전역 스코프 오염을 방지할 수 있습니다. 

{
    (function () {
        console.log("25. 실행되었습니다.");
    })();
}
{
    (() => {
        console.log("26. 실행되었습니다.");
    })();
}
결과 확인하기
25. 실행되었습니다.
26. 실행되었습니다.

21. 함수: 파라미터 함수

파라미터 함수는 다른 함수에 전달되어 사용되는 함수로, 함수 호출 시 인자(argument)로 전달됩니다. 이를 통해 함수의 동작을 유연하게 조절하고 재사용성을 높일 수 있습니다. 

{
    function func(str = "27. 실행되었습니다.") {
        console.log(str);
    }
    func();
}
{
    const func = (str = "28. 실행되었습니다.") => {
        console.log(str);
    }
    func();
}
결과 확인하기
27. 실행되었습니다.
28. 실행되었습니다.

22. 함수: 재귀 함수

재귀 함수는 자신의 함수 내에서 자신을 호출하여 작업을 수행하는 함수입니다. 이러한 방식으로 함수가 자신을 반복적으로 호출함으로써 문제를 해결하거나 작업을 수행할 수 있습니다. 

{
    // 01 반복문을 사용하는 경우
    function func(num) {
        for (let i = 0; i < num; i++) {
            console.log("29. 실행되었습니다.");
        }
    }
    func(10);

    // 02 재귀함수를 사용하기
    function func1(num) {
        if (num < 1) return;
        console.log("30. 실행되었습니다.");
        func1(num - 1); //재귀 호출
    }
    func(10);
}
결과 확인하기
[X10] 29. 실행되었습니다.
[X10] 30. 실행되었습니다.

23. 함수: 콜백 함수

콜백 함수는 다른 함수의 인자로 전달되어, 해당 함수의 실행이 완료된 후에 호출되는 함수입니다. 주로 비동기적인 작업에서 사용되며, 작업이 완료되면 콜백 함수가 실행되어 그 결과를 처리합니다. 

{
    // 01 이벤트
    function func() {
        console.log("31. 실행되었습니다.")
    }
    // let btn = '';
    addEventListener("click",func())


    // 02 함수를 다른 함수의 인자로 전달
    function func1() {
        console.log("32. 실행되었습니다.")
    }
    function func2(callback) {
        callback()
    }
    func2(func1)

    // 03 반복문을 사용하는 경우
    function func3(num){
        console.log(num+ " 실행되었습니다.")
    }
    function func4(e) {
        for (let i = 33; i <= 38; i++) {
            e(i)
        }
    }
    func4(func3);
}
결과 확인하기
31. 실행되었습니다.
32. 실행되었습니다.
28. 실행되었습니다.
29. 실행되었습니다.
30. 실행되었습니다.
31. 실행되었습니다.
32. 실행되었습니다.

24. 함수: 비동기 함수: 콜백 함수

비동기 함수는 작업이 완료될 때까지 기다리지 않고 다음 코드를 실행하는 함수입니다. 이때 비동기 함수의 작업이 완료되면 콜백 함수가 호출되어 결과를 처리합니다. 

{
    // 동기: 작업이 순서대로, 코드는 위에서 아래로 순차진행, 작업완료 후 다음 작업시작
    // 비동기: 작업이 순서와 관계X, 대기 없이 다른작업 가능, 여러 작업 처리(ex: 콜백함수, 프로미스, async/await 등)

    // 01 동기적인 함수 호출
    {
        function func1() {
            console.log("34. 실행되었습니다.");
        }
        function func2() {
            console.log("35. 실행되었습니다.");
        }
    }
    func1();
    func2();

    // 02 비동기적인 함수 호출
    {
        function func3() {
            setTimeout(() => {
                console.log("36. 실행되었습니다.");
            }, 1000)
        }
        function func4() {
            console.log("37. 실행되었습니다.");
        }
    }
    func3();
    func4();

    // 03 콜백함수 비동기 호출
    {
        function func5(callback) {
            setTimeout(() => {
                console.log("38. 실행되었습니다.");
                callback();
            }, 1000);
        }
        function func6() {
            console.log("39. 실행되었습니다.");
        }
        func5(function () {
            func6();
        });
    }
}
결과 확인하기
34. 실행되었습니다.
35. 실행되었습니다.
37. 실행되었습니다.
36. 실행되었습니다.
38. 실행되었습니다.
39. 실행되었습니다.

26. 함수: 비동기 함수: Asyne / await

비동기 함수에서 비동기 작업을 동기적으로 처리할 수 있게 해주는 키워드인 async와 await를 사용하여 코드를 간결하고 이해하기 쉽게 만들 수 있습니다. 

{
    // 01 일반적인 함수(동기방식)
    function func() {
        console.log("42. 실행되었습니다.");
    }
    func();

    // 02 함수(비동기방식)
    async function func2() {
        console.log("43. 실행되었습니다.");
    }
    func2();

    // 03 비동기 방식(fetch())
    async function func3() {
        const result = await fetch("https://gnlgk.github.io/class2024/json/gineungsaJC2005_02.json");
        const data = await result.json();
        console.log(data);
    }
    func3();

    // 04 비동기 + 에러
    async function func4() {
        try {
            const result = await fetch("https://gnlgk.github.io/class2024/json/gineungsaJC2005_02.json");
            const data = await result.json();
            console.log(data);
        } catch (error) {
            console.error(error);
        }
    }
    func4();
}
결과 확인하기
42. 실행되었습니다.
43. 실행되었습니다.
