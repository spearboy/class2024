01. 함수 : 선언적 함수

선언적 함수는 function 키워드를 사용하여 함수를 선언하고, 함수의 이름과 매개변수를 명시하여 코드 블록 내에서 원하는 동작을 수행하는 자바스크립트 함수입니다.

{
    function test() {
        console.log('1st function is running')
    }
    test()
}
결과 확인하기
'1st function is running'

01. 함수 : 익명 함수

익명 함수는 함수의 이름이 없는 함수로, 함수를 변수에 할당하거나 다른 함수의 매개변수로 전달할 때 사용되며, 일회성으로 사용될 때 특히 유용합니다.

{
const test = function () {
    console.log('2nd function is running')
}
test()
}
결과 확인하기
'2nd function is running'

01. 함수 : 매개변수 함수

매개변수 함수는 다른 함수의 매개변수로 전달되는 함수로, 함수 내에서 호출되거나 다른 함수로 반환될 수 있으며, 주로 콜백 함수로 사용됩니다. 이를 통해 함수의 동작을 다양하게 조절할 수 있습니다.

{
    function test(str) {
        console.log(str)
    }
    test('3rd function is running')
}
결과 확인하기
'3rd function is running'

01. 함수 : 매개변수 함수

매개변수 함수는 다른 함수의 매개변수로 전달되는 함수로, 함수 내에서 호출되거나 다른 함수로 반환될 수 있으며, 주로 콜백 함수로 사용됩니다. 이를 통해 함수의 동작을 다양하게 조절할 수 있습니다.

{
    function test() {
        return '4th function is running'
    }
    console.log(test())
}
결과 확인하기
'4th function is running'

01. 함수 : 매개변수 + 리턴값 함수

매개변수와 리턴값을 가지는 함수는 입력값(매개변수)을 받아들이고, 이를 처리하여 결과를 반환(리턴값)하는 함수로, 여러 곳에서 재사용되거나 다른 함수에 전달될 수 있습니다.

{
    function test(str) {
        return str
    }
    console.log(test('5th function is running'))
}
결과 확인하기
'5th function is running'

02. 화살표 함수 : 선언적 함수

화살표 함수는 function 키워드 대신 => 화살표를 사용하여 함수를 선언하는 축약형으로, 함수의 이름과 return 키워드를 생략하여 간결하게 함수를 정의하는 자바스크립트 함수입니다.

{
    test = () => {
        console.log('6th function is running')
    }
    test()
}
결과 확인하기
'6th function is running'

02. 화살표 함수 : 익명 함수

화살표 함수는 function 키워드 대신 => 화살표를 사용하여 함수를 선언하는 익명 함수로, 일회성으로 사용되거나 다른 함수의 매개변수로 전달될 때 특히 유용합니다.

{
    const test = () => {
        console.log('7th function is running')
    }
    test()
}
결과 확인하기
'7th function is running'

02. 화살표 함수 : 매개변수 함수

화살표 함수는 매개변수를 받아들이고 => 화살표를 통해 함수의 구현을 정의하는 함수로, 주로 다른 함수의 매개변수로 전달되는 콜백 함수나 간단한 함수를 정의할 때 사용됩니다.

{
    test = (str) => {
        console.log(str)
    }
    test('8th function is running')
}
결과 확인하기
'8th function is running'

02. 화살표 함수 : 리턴값 함수

화살표 함수는 화살표 오른쪽에 반환하려는 값을 적어주면, 그 값이 함수의 반환값으로 설정되며, 함수 본문을 한 줄로 간결하게 작성할 수 있는 자바스크립트 함수입니다.

{
    test = () => {
        return '9th function is running'
    }
    console.log(test())
}
결과 확인하기
'9th function is running'

02. 화살표 함수 : 매개변수 + 리턴값 함수

화살표 함수는 매개변수를 받아들이고, 화살표 오른쪽에 반환하려는 값을 적어주어 함수의 반환값을 설정하는 자바스크립트 함수입니다. 코드를 간결하게 작성할 수 있어 매우 유용합니다. 

{
    test = (str) => {
        return str
    }
    console.log(test('10th function is running'))
}
결과 확인하기
'10th function is running'
